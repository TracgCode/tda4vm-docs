8.6. FAQ - Supported Cores by example applications¶

8.6.1. Introduction¶

PDK provides multiple example applications that demonstrate use of driver/module. The list below details binaries and compute cores that its expected to be hosted on.

Note

There are separate tables for different revisions of SoC, however, the binaries are not specific to silicon revision.

Note that these tables are generated based on last test cycle and all bins will not be tested on all revisions of the silicon, in every test cycle.
